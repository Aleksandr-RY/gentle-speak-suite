

# Ограничение ввода в поле "Возраст ребёнка" только цифрами

## Что будет сделано

Поле "Возраст ребёнка" в форме записи будет настроено так, чтобы принимать только цифры. На мобильных устройствах будет показываться цифровая клавиатура.

## Технические детали

**Файл:** `src/components/BookingForm.tsx`

Изменения в поле `child_age`:
- Добавить `inputMode="numeric"` -- на мобильных устройствах будет показываться цифровая клавиатура
- Добавить фильтрацию ввода в `onChange` -- убирать все нецифровые символы
- Обновить placeholder с "5 лет" на "5" для ясности
